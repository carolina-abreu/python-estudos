    ## Aula 24 -  Loop for
        loop - estrutura de repetição
        for - uma das estruturas de repetição
        exemplo:
        for item in interavel:
            //execução do loop
        utilizamos loops para iterar sobre sequências ou sobre valores iteráveis
        - exemplo:
        nome = 'Geek University'
        lista = [1, 3, 5, 7, 9]
        numeros = range(1,10)
        1. iterando sobre uma string
        for letra in nome:
            print(letra) #imprime uma letra embaixo da outra
        2. iterando sobre uma lista
        for numero in lista:
            print(numero) #imprime os números da variável lista
        3. iterando sobre um range:
        for numero in range(1,10):
            print(numero) #imprime os números de 1 a 9, não imprime o 10 pq não é inclusive, o valor final do range não inclui
